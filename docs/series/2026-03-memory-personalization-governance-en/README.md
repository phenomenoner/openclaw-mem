## Core thesis
Persistent memory in LLMs fails not because systems remember too much, but because they lack the restraint to stay silent when remembering is irrelevant.

## Positioning
This series argues that the real problem in AI memory is not just storage or forgetting. It is the full governance stack around what gets written, what gets retrieved, what gets surfaced, and what should remain unsaid.
