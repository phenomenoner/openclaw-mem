## Load-bearing Claude pushes
- The four-part failure model is the strongest original contribution.
- Retention vs activation is the most valuable structural distinction.
- Attention competition should be stated explicitly: memory injected into context competes with the current task.
- The no-memory baseline matters; persistent memory must justify its cost and risk.
- Part 3 should be honest about current proof limits.
- Part 4 should open with the evaluation / counterfactual problem.
- Sharp line:
  - `A correct recall that the user didn't ask for is indistinguishable from a wrong recall.`
